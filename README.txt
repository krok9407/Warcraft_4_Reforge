Это учебный курс для студентов

Для запуска игры откройте файл Start.html (для корректной работы всех функций используйте браузер Google Chrome)

Управление:
	Передвижение персонажа
		Влево - ← 
		Вправо - →
		Прыжок - ↑
	Стрельба - a (англ.)

Игра сделана по мотивам культовой вселенной Warcraft.

Вы играете за Иллидана Ярость Бурь. Вам необходимо дойти до последнего уровня, собирая магию, хранившуюся в черепе Гул'дана. Либо одолейте соперников на каждом из уровней при помощи заклинаний.

1 уровень - Король Лич (локация - Цитадель ледяной короны)
2 уровень - Магтеридон (локация - Награнд)
3 уровень - Майев Песнь Теней (локация - Черный Храм)

5 собранных черепов позволят заполнить полоску опыта для перехода на новый уровень, либо доведите индикатор жизни соперника до минимума.
Следите за индикатором вашего здоровья, иначе вам придется перенестись из мира духов в новое тело и начать прогресс заново.

Не забудьте ввести свой никнейм и наслаждаться звуковым сопровождением на каждом из уровней.
